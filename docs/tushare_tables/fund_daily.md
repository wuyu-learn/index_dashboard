# ETF 日线行情

## 接口信息

| 项目 | 内容 |
| --- | --- |
| 接口名称 | `fund_daily` |
| 描述 | 获取 ETF 每日收盘后的成交行情，历史数据超过 10 年 |
| 调取限量 | 单次最多返回 5,000 行，可按 ETF 代码和日期循环获取，历史总量不限 |
| 积分要求 | 至少需要 5,000 积分；8,000 积分可获得更高调取频次 |
| 积分说明 | [Tushare 积分获取办法](https://tushare.pro/document/1?doc_id=13) |

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| `ts_code` | `str` | 否 | 基金代码 |
| `trade_date` | `str` | 否 | 交易日期，格式为 `YYYYMMDD` |
| `start_date` | `str` | 否 | 开始日期，格式为 `YYYYMMDD` |
| `end_date` | `str` | 否 | 结束日期，格式为 `YYYYMMDD` |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
| --- | --- | --- | --- |
| `ts_code` | `str` | 是 | TS 基金代码 |
| `trade_date` | `str` | 是 | 交易日期 |
| `open` | `float` | 是 | 开盘价（元） |
| `high` | `float` | 是 | 最高价（元） |
| `low` | `float` | 是 | 最低价（元） |
| `close` | `float` | 是 | 收盘价（元） |
| `pre_close` | `float` | 是 | 昨日收盘价（元） |
| `change` | `float` | 是 | 涨跌额（元） |
| `pct_chg` | `float` | 是 | 涨跌幅（%） |
| `vol` | `float` | 是 | 成交量（手） |
| `amount` | `float` | 是 | 成交额（千元） |

## 接口示例

获取“沪深 300 ETF 华夏”（`510330.SH`）2025 年以来的行情，并通过 `fields` 指定返回字段：

```python
import tushare as ts

pro = ts.pro_api()

df = pro.fund_daily(
    ts_code="510330.SH",
    start_date="20250101",
    end_date="20250618",
    fields="trade_date,open,high,low,close,vol,amount",
)
```

## Demo 阶段 CSV 存储约定

- 建议路径：`data/raw/fund_daily/YYYY/YYYYMMDD.csv`
- 文件粒度：每个交易日一个文件，保存当日全部目标 ETF
- 唯一键：`ts_code + trade_date`
- 日期字段保留为 `YYYYMMDD` 字符串
- 价格、成交量和成交额保留 Tushare 原始单位
- 重新拉取同一交易日时覆盖原文件
