# ETF 份额规模

## 接口信息

| 项目 | 内容 |
| --- | --- |
| 接口名称 | `etf_share_size` |
| 描述 | 获取沪深 ETF 每日份额和规模数据，用于观察规模、份额及资金动向，同时提供每日净值和收盘价 |
| 更新时间 | 数据分批入库，交易所通常在次日约 08:30 更新上一交易日数据；涉及海外资产的 ETF 更新可能更晚 |
| 调取限量 | 单次最多返回 5,000 条数据，可按 ETF 代码或日期循环提取 |
| 积分要求 | 用户达到 8,000 积分后可调取 |
| 积分说明 | [Tushare 积分获取办法](https://tushare.pro/document/1?doc_id=13) |

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| `ts_code` | `str` | 否 | 基金代码，可从 ETF 基础信息接口获取 |
| `trade_date` | `str` | 否 | 交易日期，格式为 `YYYYMMDD` |
| `start_date` | `str` | 否 | 开始日期，格式为 `YYYYMMDD` |
| `end_date` | `str` | 否 | 结束日期，格式为 `YYYYMMDD` |
| `exchange` | `str` | 否 | 交易所：`SSE` 上交所、`SZSE` 深交所 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
| --- | --- | --- | --- |
| `trade_date` | `str` | 是 | 交易日期 |
| `ts_code` | `str` | 是 | ETF 代码 |
| `etf_name` | `str` | 是 | 基金名称 |
| `total_share` | `float` | 是 | 总份额，单位为万份 |
| `total_size` | `float` | 是 | 总规模，单位为万元 |
| `nav` | `float` | 否 | 基金份额净值，单位为元 |
| `close` | `float` | 否 | 收盘价，单位为元 |
| `exchange` | `str` | 是 | 交易所：`SSE` 上交所、`SZSE` 深交所、`BSE` 北交所 |

## 接口示例

```python
# 获取“沪深 300 ETF 华夏”2025 年以来每个交易日的份额和规模
df = pro.etf_share_size(
    ts_code="510330.SH",
    start_date="20250101",
    end_date="20251224",
)

# 获取 2025 年 12 月 24 日上交所全部 ETF 的份额和规模
df = pro.etf_share_size(
    trade_date="20251224",
    exchange="SSE",
)
```

## Demo 阶段 CSV 存储约定

- 建议路径：`data/raw/etf_share_size/YYYY/YYYYMMDD.csv`
- 文件粒度：每个交易日一个文件，保存当日全部目标 ETF
- 唯一键：`ts_code + trade_date`
- 日期字段保留为 `YYYYMMDD` 字符串
- `total_share`、`total_size`、`nav` 和 `close` 保留 Tushare 原始单位与精度
- 不将缺失的 `nav` 或 `close` 填充为 `0`
- `exchange` 保留接口返回的 `SSE`、`SZSE` 或 `BSE`

## 同步建议

- 不建议在交易日收盘后立即获取当日数据。
- 建议在下一交易日 `09:00` 后同步上一交易日数据。
- 每次重新获取最近 5 个交易日并覆盖对应 CSV，以补齐分批入库和延迟数据。
- 海外资产 ETF 可能更新较晚，可在当日下午或再下一日进行二次补采。
- 如果按日期获取超过 5,000 条限制，应按 `exchange` 分批调用。
