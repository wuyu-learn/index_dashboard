# 指数基本信息

## 接口信息

| 项目 | 内容 |
| --- | --- |
| 接口名称 | `index_basic` |
| 描述 | 获取指数基础信息 |
| 在线调试 | [Tushare 数据工具](https://tushare.pro/webclient/) |

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| `ts_code` | `str` | 否 | TS 指数代码 |
| `symbol` | `str` | 否 | 指数代码，支持多值输入，例如 `000300,000001` |
| `name` | `str` | 否 | 指数简称 |
| `market` | `str` | 否 | 交易所或服务商，默认为 `SSE`，取值参见市场代码说明 |
| `publisher` | `str` | 否 | 发布商 |
| `category` | `str` | 否 | 指数类别 |

## 输出参数

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| `ts_code` | `str` | TS 指数代码 |
| `name` | `str` | 指数简称 |
| `fullname` | `str` | 指数全称 |
| `market` | `str` | 市场 |
| `publisher` | `str` | 发布方 |
| `index_type` | `str` | 指数风格 |
| `category` | `str` | 指数类别 |
| `base_date` | `str` | 基期 |
| `base_point` | `float` | 基点 |
| `list_date` | `str` | 发布日期 |
| `weight_rule` | `str` | 加权方式 |
| `desc` | `str` | 描述 |
| `exp_date` | `str` | 终止日期 |

## 市场代码说明

| 市场代码 | 说明 |

## 接口示例

```python
import tushare as ts

pro = ts.pro_api()

# 获取申万指数基础信息
df = pro.index_basic(market="SW")
```

## Demo 阶段 CSV 存储约定

- 建议路径：`data/raw/index_basic/index_basic.csv`
- 文件粒度：保存目标市场的全部指数基础信息
- 唯一键：`ts_code`
- 日期字段 `base_date`、`list_date` 和 `exp_date` 保留为 `YYYYMMDD` 字符串
- `base_point` 保留 Tushare 返回的原始数值精度
- 不将缺失的终止日期 `exp_date` 填充为虚构日期
- 因接口默认只查询 `SSE`，全量同步时应分别按所需 `market` 调用、合并并按 `ts_code` 去重
- `etf_index` 是 ETF 当前跟踪的基准指数列表，`index_basic` 是更广泛的指数基础资料，两者不可相互替代
