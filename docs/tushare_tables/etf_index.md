# ETF 基准指数列表

## 接口信息

| 项目 | 内容 |
| --- | --- |
| 接口名称 | `etf_index` |
| 描述 | 获取 ETF 基准指数列表信息 |
| 调取限量 | 单次最多返回 5,000 行数据，当前指数总数未超过 2,000 |
| 积分要求 | 用户达到 8,000 积分后可调取 |
| 积分说明 | [Tushare 积分获取办法](https://tushare.pro/document/1?doc_id=13) |

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| `ts_code` | `str` | 否 | 指数代码 |
| `pub_date` | `str` | 否 | 发布日期，格式为 `YYYYMMDD` |
| `base_date` | `str` | 否 | 指数基期，格式为 `YYYYMMDD` |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
| --- | --- | --- | --- |
| `ts_code` | `str` | 是 | 指数代码 |
| `indx_name` | `str` | 是 | 指数全称 |
| `indx_csname` | `str` | 是 | 指数简称 |
| `pub_party_name` | `str` | 是 | 指数发布机构 |
| `pub_date` | `str` | 是 | 指数发布日期 |
| `base_date` | `str` | 是 | 指数基日 |
| `bp` | `float` | 是 | 指数基点，单位为点 |
| `adj_circle` | `str` | 是 | 指数成份证券调整周期 |

## 接口示例

```python
# 获取当前 ETF 跟踪的基准指数列表
df = pro.etf_index(
    fields="ts_code,indx_name,pub_date,bp",
)
```

## Demo 阶段 CSV 存储约定

- 建议路径：`data/raw/etf_index/etf_index.csv`
- 文件粒度：保存全部 ETF 基准指数的最新信息
- 唯一键：`ts_code`
- 更新方式：每次全量拉取并原子覆盖 CSV
- `pub_date` 和 `base_date` 保留为 `YYYYMMDD` 字符串
- `bp` 保留 Tushare 返回的原始数值精度
- 可通过 `etf_basic.index_code = etf_index.ts_code` 关联 ETF 与基准指数
